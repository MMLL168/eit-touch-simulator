Hardware Setup

Complete guide to assembling and configuring your EIT Touch Simulator hardware

Hardware Overview

The EIT Touch Simulator hardware consists of several key components that work together to create an interactive electrical impedance tomography system. This guide will walk you through the complete assembly and configuration process.

Required Components

Microcontroller

  • Arduino Mega 2560 or compatible
  • USB cable for programming
  • Power supply (7-12V DC)

Touch Sensors

  • 16x capacitive touch sensors
  • Copper electrodes or conductive fabric
  • Shielded cables

Display & Interface

  • LCD display (optional)
  • Status LEDs
  • Reset button

Assembly Instructions

Step 1: Prepare the Circuit Board

Start by laying out your circuit board or breadboard. Ensure you have adequate space for all components and proper organization of wiring.

Tip: Use a breadboard layout planning tool to visualize your circuit before assembly.

Step 2: Install the Microcontroller

Mount the Arduino Mega 2560 securely on your board. Ensure all pins are accessible and the USB port is easily reachable for programming.

  • Secure the Arduino using standoffs or mounting brackets
  • Verify power connections (GND and VCC)
  • Test with a simple LED blink program

Step 3: Connect Touch Sensors

Wire the capacitive touch sensors to the Arduino's analog inputs. Use shielded cables to minimize electromagnetic interference.

Pin Mapping:
Sensor 1-8: A0-A7
Sensor 9-16: A8-A15
Ground: Common GND

Step 4: Install Electrodes

Arrange the 16 electrodes in a circular pattern around the sensing area. Ensure even spacing and secure mounting.

  • Recommended diameter: 20-30 cm
  • Electrode spacing: 22.5° apart (360°/16)
  • Use conductive adhesive or soldering

Step 5: Wire the Display (Optional)

Connect an LCD display for real-time status monitoring and debugging information.

I2C LCD Connections:
SDA: Pin 20
SCL: Pin 21
VCC: 5V
GND: GND

Step 6: Add Status LEDs

Install status LEDs to indicate power, data transmission, and error states.

  • Power LED (Green): Pin 13 through 220Ω resistor
  • Activity LED (Blue): Pin 12 through 220Ω resistor
  • Error LED (Red): Pin 11 through 220Ω resistor

Calibration & Testing

Initial Calibration

  1. Power on the system without any touch input
  2. Allow 30 seconds for baseline stabilization
  3. Run the calibration routine from the software
  4. Test each electrode individually
  5. Verify signal quality and noise levels

Testing Procedures

  1. Perform single-touch tests on each electrode
  2. Test multi-touch detection (2-5 simultaneous touches)
  3. Verify data transmission to computer
  4. Check visualization output
  5. Document any anomalies or issues

Troubleshooting

Common Issues and Solutions

Problem: Sensors not responding

Solutions:

  • Check all wire connections and solder joints
  • Verify power supply voltage (should be 5V at sensor pins)
  • Ensure proper grounding of the system
  • Test sensors individually with a multimeter

Problem: Noisy or unstable readings

Solutions:

  • Use shielded cables for sensor connections
  • Add capacitors (0.1μF) across power lines
  • Increase distance from power supplies and motors
  • Implement software filtering (moving average)

Problem: Inconsistent touch detection

Solutions:

  • Recalibrate the baseline values
  • Adjust sensitivity thresholds in software
  • Check for damaged or corroded electrodes
  • Ensure proper electrode-to-skin contact

Problem: No serial communication

Solutions:

  • Verify USB cable connection
  • Check correct COM port selection in software
  • Ensure Arduino drivers are properly installed
  • Test with Arduino IDE Serial Monitor first

Safety Guidelines

⚠️ Important Safety Information

  • Electrical Safety: Always disconnect power before making circuit modifications
  • Grounding: Ensure proper grounding to prevent electric shock
  • Voltage Limits: Do not exceed 12V DC input to the Arduino
  • Heat Management: Provide adequate ventilation for all components
  • ESD Protection: Use anti-static precautions when handling electronics
  • Medical Use: This is a simulator for research/educational purposes only, not approved for medical use

Maintenance & Care

Regular Maintenance

  • Clean electrodes weekly with isopropyl alcohol
  • Inspect wiring for damage or wear monthly
  • Re-calibrate sensors if performance degrades
  • Update firmware as new versions are released
  • Keep backup of calibration settings

Storage Guidelines

  • Store in a dry, dust-free environment
  • Disconnect power when not in use for extended periods
  • Protect from direct sunlight and extreme temperatures
  • Cover sensitive components to prevent oxidation
  • Document any modifications or repairs
