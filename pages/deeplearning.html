🧠 深度學習優化

Deep Learning for Fast EIT Image Reconstruction

🚀 為何需要深度學習？

傳統方法的挑戰

  • Newton-Raphson 迭代慢 (100-500ms)
  • 需要正則化參數調整
  • 對雜訊敏感
  • 難以即時應用

深度學習的優勢

  • 前饋推論快速 (< 10ms)
  • 端到端學習，無需手動調參
  • 自動學習去噪
  • 支援即時互動

🏗️ UNet 網路架構

UNet 是一種編碼器-解碼器架構，特別適合影像重建任務。

UNet 架構：編碼器降維、解碼器升維、跳躍連接保留細節

📚 訓練策略

1. 數據生成

合成數據：使用 FEM 正向求解器生成訓練數據

  • 隨機生成觸控位置 (10,000 筆)
  • 計算對應的邊界電壓
  • 添加高斯雜訊模擬實際測量

2. 損失函數

組合損失：平衡影像品質與位置精度

Loss = MSE(image) + 0.1 × L1(position) + 0.01 × SSIM(image)

3. 訓練參數

參數 值
優化器Adam (lr=0.001)
Batch Size32
Epochs100
訓練時間約 2 小時 (GPU)

🌐 TensorFlow.js 整合

將訓練好的模型部署到瀏覽器，實現客戶端即時推論。

部署流程

  1. 使用 Python TensorFlow 訓練模型
  2. 轉換為 TensorFlow.js 格式 (tfjs_converter)
  3. 載入到瀏覽器 (model.json + weights.bin)
  4. 前端 JavaScript 調用推論

範例程式碼

// 載入模型
const model = await tf.loadGraphModel('model/model.json');

// 推論
const input = tf.tensor2d(voltages, [1, 380]);
const output = model.predict(input);
const reconstruction = output.dataSync();
